A simple Python game where the player tries to guess a random number.

## How it works
- The computer chooses a random number between 1 and 100
- The player has 5 attempts to guess the number
- The program tells the player if the guess is higher or lower
- Duplicate guesses are not allowed

## How to run
1. Make sure Python is installed
2. Run the program with:
   python main.py

## Features
- Limited attempts
- Input validation for duplicate numbers
- Helpful hints (higher / lower)
